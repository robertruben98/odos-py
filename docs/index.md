# odos-py

Modern Python client for the [Odos](https://www.odos.xyz/) DEX Aggregator API —
sync + async, quote → assemble flow, typed with Pydantic v2.

## Install

```bash
pip install odos-py
```

The distribution is `odos-py`; the import package is `odos_py`.

## Quick start

```python
import odos_py
```

See the [API reference](api.md) for the full surface.
